II. PROJECT DNA

Emotion: Nostalgia. Seduction. The beauty of imperfection — dust on the lens, grain in the shadow, the warmth of analog. Like finding a photograph in a book you forgot you owned.


World: A Rome screening room. 35mm projector light cutting through cigarette smoke. The flicker of film. A Fellini film you've never seen but somehow remember.


Archetype: Cinematic Glamour


Audience Fantasy: The cinephile, the analog romantic, the person who still buys physical photographs. Someone who believes beauty was better when it was imperfect.


North Star Moment: A film leader countdown (5... 4... 3... 2... 1...) plays on a black screen, complete with scratches and dust. Then the "film" starts — the first image flickers with real film gate jitter.

V. SITE ARCHITECTURE

THE ENTIRE SITE IS HORIZONTAL. Vertical scroll drives horizontal movement. The viewport is a film strip.


SCENE 00 — FILM LEADER
Countdown: 5... 4... 3... 2... 1... white numbers on black, scratched, with film dust particles. Each number holds 0.6s. After "1", the film "starts" — the first frame flickers in.


SCENE 01 — HERO FRAME
The first frame of the "film." Elena's portrait, full-bleed, with sprocket hole decorations on top and bottom edges. Her name in Instrument Serif overlaid.


SCENE 02 — THE REEL (EDITORIAL)
8 frames, each separated by sprocket holes. Horizontal scroll moves through them frame by frame. Each frame holds for the "duration" of scroll distance. Frame counter: 001 / 008.


SCENE 03 — INTERMISSION
A title card: white text on black — the manifesto/quote section. "Cinema is truth twenty-four times per second." Holds for 3 seconds of scroll distance.


SCENE 04 — THE SECOND REEL (CAMPAIGNS)
4 campaign frames, larger than editorial.


SCENE 05 — THE ARCHIVE
Text list scrolling horizontally. On hover: frame behind the text develops (image fades in).


SCENE 06 — CREDITS
The end credits. Text scrolls up vertically (traditional film credit roll). Her name. The agencies. The footer.



VI. SIGNATURE INTERACTION — FILM GATE JITTER

Every image has a constant subtle CSS animation mimicking film gate jitter:

css
css
@keyframes jitter {
  0%   { transform: translate(0, 0); }
  25%  { transform: translate(-1px, 1px); }
  50%  { transform: translate(1px, -1px); }
  75%  { transform: translate(-1px, 0); }
  100% { transform: translate(0, 1px); }
}
Running at 24fps (each step 41.6ms). On hover, the jitter STOPS — the "projector focuses." The image becomes stable. Film dust particles float across the viewport as a CSS pseudo-element.



VII. SCROLL

Vertical-to-horizontal GSAP translation. Lerp 0.06. Slow, frame-by-frame feeling. Each frame gets equal scroll distance.
